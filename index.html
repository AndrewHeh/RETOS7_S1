PRE PROCESADOR - CSS

INTRODUCCION:

Viene a ser un pre procesador, SASS, este le proviene al css el termino "escalable"

PREPROCESADOR CSS :

Programa que permite generar CSS a partir de la syntaxis propia o unica de este, hay variedades de estos pero los mas destacados son, Less, Sass, Stylus.

SASS : QUE ES?

Herramienta que nos permite generar, de manera automática, hojas de estilo, añadiéndoles características que no tiene CSS.

VENTAJAS

El usar esta herramienta proporciona una serie de ventajas:

  • Reduce el tiempo para crear y mantener el CSS.
  • Proporciona estructuras avanzadas propias de los lenguajes de programación, como variables, listas, funciones y estructuras de control.
  • Permite tener una organización modular de los estilos, lo cual es vital para proyectos grandes.
    •

    DESVENTAJAS:

    Teniendo claro que "SASS" no viene a ser un lenguaje de programacion, viene a ser complemento limitado y por lo tanto tiene sus desventajas:

  • Claramente viene a ser una nueva herramienta, para algunos suponen que esto es una desventaja
  • Tiene una sintaxis mas compleja que CSS
